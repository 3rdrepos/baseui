Packagecom.soma.ui.layouts
Classpublic class CanvasUI
InheritanceCanvasUI LayoutUI flash.display.MovieClip

The CanvasUI class extends the LayoutUI class and works in the same way, but the children in the display list of the canvas are automatically added to an internal BaseUI instance. The CanvasUI class provides the necessary methods to get the ElementUI instances of its children.

Examples
     var canvas:CanvasUI = new CanvasUI(stage, 400, 300);
     canvas.backgroundColor = 0xFF0000;
     canvas.backgroundAlpha = 0.2;
     canvas.ratio = ElementUI.RATIO_IN;
     addChild(canvas);
     
     var sprite:Sprite = new Sprite();
     sprite.graphics.beginFill(0xFF0000, .5);
     sprite.graphics.drawRect(0, 0, 100, 100);
     
     var element:ElementUI = canvas.add(sprite);
     element.right = 10;
     element.bottom = 10;
         
     canvas.refresh();
